# Game Base

A foundational JavaScript/HTML5 Canvas game engine with basic structure and mechanics.

## Features

- **Game Loop**: Smooth 60 FPS game loop with delta time calculation
- **Player Control**: Move the player with Arrow Keys or A/D
- **Input Handling**: Responsive keyboard input system
- **Score System**: Basic score tracking
- **FPS Display**: Real-time FPS counter
- **Boundary Detection**: Player collision with canvas edges

## Project Structure

```
K/
├── index.html      # HTML structure and canvas
├── styles.css      # Styling and layout
├── game.js         # Core game logic and loop
└── README.md       # This file
```

## How to Run

1. Open `index.html` in a web browser
2. Use **Arrow Keys** or **A/D** to move the player
3. Watch your score increase!

## Game Objects

### Player
- Position: Center-bottom of screen
- Size: 40x40 pixels
- Color: Red
- Movement: Left/Right using Arrow Keys or A/D
